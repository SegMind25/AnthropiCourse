## 🧠 Key Concepts I'm Learning

### 🎯 The 3 Elements of a Great Prompt

Before talking to Claude, I now think about **3 key things**:

| Element | Question to Ask Yourself | Example |
|---|---|---|
| **Setting the Stage** | What's my role & context? | *"I'm a marketing lead at a startup..."* |
| **Defining the Task** | What action do I want? | *"...research the streaming market..."* |
| **Specifying Rules** | What style/format do I need? | *"...structured as a professional report."* |

### 💡 The "Coworker Approach"
The best way to talk to Claude is **naturally, concisely, and conversationally** — like messaging a smart coworker, not typing a Google search.

### 🔄 Iterating on Responses
Claude conversations are meant to go **back and forth**. If the first answer isn't perfect:
- Ask for more detail → *"Can you expand on point 2?"*
- Give feedback → *"Good, but make it more casual."*
- Redirect → *"Actually, I meant X, not Y."*

### 📎 Adding Context with Uploads
Claude can read and analyze:
- 📄 Documents: PDF, DOCX, TXT, CSV
- 🖼️ Images: PNG, JPEG
- 💻 Code files

This is a shortcut to give Claude **your world** so it can give better answers.

## 🔬 Practice Example

Here's a prompt I built using the 3-element framework:

```
"I'm the marketing lead at an indie streaming startup, and we're preparing 
an investor pitch deck. Can you research the current state of the independent 
film streaming market and identify key trends, competitor positioning, and 
growth opportunities? Use current web research with citations and structure 
it as a professional report."
```

✅ **Stage set** → indie streaming startup, investor pitch context  
✅ **Task defined** → research market trends, competitors, opportunities  
✅ **Rules specified** → web citations, professional report format

## 🚀 The 4D Framework for AI Fluency

This course is adapted from research by **Professor Rick Dakan** and **Professor Joseph Feller**. The framework has 4 core competencies:

1. **Delegation** — Knowing *what* to hand off to AI
2. **Description** — Knowing *how* to explain your request clearly
3. **Discernment** — Knowing *when* the AI's answer is good enough
4. **Diligence** — Knowing *how* to verify and refine results
