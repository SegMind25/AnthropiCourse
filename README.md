## 👋 About This Repository

This repo documents my hands-on journey through **Anthropic's official AI learning platform** — covering everyday Claude use, AI fluency principles, developer tools, and real-world projects built with Claude Code.

Each folder is a self-contained module with structured notes, key concepts, and project files — organized as both a personal reference and a resource for anyone starting their AI collaboration journey.

> *"AI Fluency is the ability to collaborate with AI effectively, efficiently, ethically, and safely."*

## 🏆 The 4D Framework for AI Fluency

The foundation of everything — developed by **Prof. Rick Dakan** & **Prof. Joseph Feller**:

| Competency | Core Question | What It Covers |
|---|---|---|
| 🎯 **Delegation** | *Should I give this to AI?* | Deciding what humans do vs. what AI does |
| 📝 **Description** | *Am I explaining this clearly?* | Communicating effectively with AI |
| 🔍 **Discernment** | *Is this output good enough?* | Critically evaluating AI responses |
| 🛡️ **Diligence** | *Am I being responsible?* | Using AI ethically and safely |
