## 🏆 The 4D Framework for AI Fluency

The foundation of everything in this course — developed by **Prof. Rick Dakan** & **Prof. Joseph Feller**:

| Competency | Core Question | What It Covers |
|---|---|---|
| 🎯 **Delegation** | *Should I give this to AI?* | Deciding what humans do vs. what AI does |
| 📝 **Description** | *Am I explaining this clearly?* | Communicating effectively with AI |
| 🔍 **Discernment** | *Is this output good enough?* | Critically evaluating AI responses |
| 🛡️ **Diligence** | *Am I being responsible?* | Using AI ethically and safely |

## 🧙 Key Concepts at a Glance

**Prompting**
- Set the stage → Define the task → Specify the rules
- Iterate: follow-up, give feedback, redirect or restart

**Claude's Modes** *(Desktop App)*
- 💬 Chat = quick conversations
- 🤝 Cowork = deep research & complex documents
- 🖥️ Code = software development

**Projects vs Skills**
- 📁 Projects = *what* Claude knows (knowledge hub)
- 🛠️ Skills = *how* Claude works (process engine)

**Connectors & MCP**
- MCP = universal standard (like USB-C) for connecting Claude to any tool
- Web Connectors = cloud apps · Desktop Extensions = local apps

**Research Mode**
- Multi-step investigation · 5–45 min · auto-cites all sources
- Extended thinking enabled automatically
