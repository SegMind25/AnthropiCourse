## 🧠 Lesson 1 — Your First Conversation with Claude

### 🎯 The 3 Elements of a Great Prompt

| Element | Question to Ask Yourself | Example |
|---|---|---|
| **Setting the Stage** | What's my role & context? | *"I'm a marketing lead at a startup..."* |
| **Defining the Task** | What action do I want? | *"...research the streaming market..."* |
| **Specifying Rules** | What style/format do I need? | *"...structured as a professional report."* |

### 💡 The "Coworker Approach"
Talk to Claude **naturally, concisely, and conversationally** — like messaging a smart coworker, not typing a Google search.

### 🔄 Iterating on Responses
- Ask for more detail → *"Can you expand on point 2?"*
- Give feedback → *"Good, but make it more casual."*
- Redirect → *"Actually, I meant X, not Y."*

## 🎯 Lesson 2 — Getting Better Results

### 🐛 Common Challenges & How to Fix Them

| Problem | Why It Happens | The Fix |
|---|---|---|
| **Too generic** | Not enough context | Add your role, audience & constraints |
| **Too long / too short** | Claude is guessing at length | Be explicit: *"Keep this under 100 words"* |
| **Wrong format** | Claude knows WHAT but not HOW | Show an example or describe the structure |
| **Wrong information** | Claude can occasionally be wrong | Verify facts independently for high-stakes work |
| **Wrong tone** | Claude defaults to professional | Say exactly what you want: *"More conversational"* |

### 🔁 The Iteration Mindset
> *"Your first prompt is the start of a conversation, not a one-shot request."*

✅ Treat first drafts as **starting points**  
✅ Give **specific feedback** → *"Cut the first two paragraphs"* not just *"Make it shorter"*  
✅ Know when to **start fresh** — sometimes a new chat is faster  

### 🏆 The 4D Framework for AI Fluency

| Competency | What It Means |
|---|---|
| 🎯 **Delegation** | Decide what AI does vs. what humans do |
| 📝 **Description** | Communicate clearly with AI |
| 🔍 **Discernment** | Critically evaluate AI outputs |
| 🛡️ **Diligence** | Use AI responsibly & ethically |

## 💻 Lesson 3 — Claude Desktop App: Chat, Cowork & Code

The Claude desktop app has **3 powerful modes**, each designed for a different type of work.

---

### 💬 Mode 1 — Chat

> Best for: quick questions, brainstorming, drafting, back-and-forth conversations

![Chat Mode](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/69a8c30634a6c3e545269791_69a8a456caa28723041a6eea_699e1b746d83c9fb55a56d31_698be3352261cf694fdae308_698bdd295c5f5d680c01e63c_Screenshot%25252525202026-02-09%2525252520at%25252525204.38.22%25252525E2%2525252580%25252525AFPM.png)

**Key Features:**
- ⚡ **Quick Entry** — Double-tap `Option` key (Mac) to pull up Claude over any app
- 📸 **Screenshots & Window Sharing** — Show Claude exactly what's on your screen
- 🎙️ **Dictation** — Talk through a problem instead of typing
- 🔌 **Desktop Connectors** — Connect local tools and services

---

### 🤝 Mode 2 — Cowork

> Best for: deep research, multi-source analysis, complex documents, recurring tasks

![Cowork Mode](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/69a8c30634a6c3e54526979c_69a8a456caa28723041a6ee7_699e2c4828218e71872531df_699e289b7bffb76b5ebedc79_Screenshot%252525202026-02-24%25252520at%252525202.39.06%252525E2%25252580%252525AFPM.png)

**Key Features:**
- 📁 **Folder Access** — Give Claude a folder; it reads, analyzes & saves work there
- ⏰ **Scheduled Tasks** — Set recurring tasks (daily briefing, weekly roundup, inbox triage)
- 🌐 **Browser Use** — Claude can navigate websites and pull data automatically
- 🧩 **Plugins** — Add live financial data, internal knowledge bases, and more
- 🔒 **Protected Environment** — Claude only accesses folders you explicitly share

> ⚠️ *Currently a research preview — available to Pro, Max, Team & Enterprise users*

---

### 🖥️ Mode 3 — Code

> Best for: writing, testing, running, and deploying software

![Code Mode](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/69a8c30634a6c3e545269794_69a8a455caa28723041a6ee1_699e1b736d83c9fb55a56d2a_698be3342261cf694fdae301_698bdd78a6031117e6bb41d6_Screenshot%25252525202026-02-09%2525252520at%252525252012.52.07%25252525E2%2525252580%25252525AFPM.png)

**Key Features:**
- 👁️ **Visual Diffs** — See exactly what code changed before accepting
- 💻 **Built-in Terminal** — Commands run live as Claude works
- 🔀 **Git Integration** — Every version tracked, always rollback-able
- 🏠 **Local** — Claude works directly in your files on your machine
- ☁️ **Remote** — Connect a GitHub repo; sessions continue even if you close the app

**3 Interaction Modes:**
| Mode | What It Does |
|------|-------------|
| 🔍 **Ask** | Claude proposes every change — you approve before anything is modified |
| ⚡ **Code** | Claude applies file changes automatically, but asks before running commands |
| 📋 **Plan** | Claude outlines its full strategy before touching anything |

> ⚠️ *Rolling out to Pro, Max, Team & Enterprise users*
